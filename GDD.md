# 독련 (Dokryun) - Game Design Document

## 컨셉
- **제목**: 독련 (獨鍊) - "홀로 단련하는 자"
- **의미**: 독(獨, 홀로) + 련(鍊, 단련) → 윤회하며 홀로 싸워 강해지는 자
- **장르**: 조선풍 로그라이크 보스러시 (세피리아 류)
- **비주얼**: 도트 감성 + 2.5D (깊이감 있는 탑다운/쿼터뷰)
- **플랫폼**: Steam (Windows)
- **가격**: 3,900원
- **엔진**: MonoGame (C#)

## 게임 루프
```
[윤회 시작] → [스테이지 진입] → [쫄몹 처치] → [증강 선택] → [보스전] → [사망/클리어] → [윤회]
```

### 1라운드 흐름
1. **쫄몹 웨이브** (30~60초) - 사방에서 몰려오는 적 처치
2. **증강 선택** - 3개 중 1개 선택 (공격/방어/특수)
3. **다음 웨이브** or **보스전**
4. 보스 처치 시 → 다음 스테이지 (난이도 상승)
5. 사망 시 → 윤회 (영구 성장 요소 일부 유지)

## 세계관
- **배경**: 조선 말기, 귀신과 요괴가 들끓는 난세
- **주인공**: 윤회의 저주를 받은 무사. 죽을 때마다 다시 태어나지만, 이전 생의 기억 파편이 남음
- **목표**: 윤회의 고리를 끊기 위해 최종 보스 "업왕(業王)"을 쓰러뜨림

## 플레이어
### 기본 조작
- **WASD / 방향키**: 8방향 이동
- **마우스 좌클릭**: 기본 공격 (근접/원거리 무기 의존)
- **마우스 우클릭**: 특수기 (쿨타임)
- **Space**: 회피 (구르기, 무적 프레임)
- **Q/E**: 스킬 슬롯 1/2

### 스탯
- **체력(HP)**: 피격 시 감소, 0이면 사망 → 윤회
- **기력(氣)**: 특수기/스킬 사용 시 소모, 자동 회복
- **공격력**: 기본 데미지
- **이동속도**: 기본 이동 속도
- **치명타율**: 크리티컬 확률

## 증강 시스템
웨이브 클리어 시 3개 중 1개 선택:

### 등급
- **백(白)** - 일반 (흔함)
- **청(靑)** - 고급
- **홍(紅)** - 희귀
- **금(金)** - 전설

### 증강 예시
| 이름 | 등급 | 효과 |
|------|------|------|
| 광풍참 | 백 | 공격 범위 +15% |
| 철벽기공 | 백 | 방어력 +10 |
| 독기술 | 청 | 공격 시 독 부여 (3초간 지속 데미지) |
| 뇌전보법 | 청 | 회피 시 번개 흔적 생성, 적에게 데미지 |
| 귀신보 | 홍 | 이동속도 +30%, 잔상 생성 |
| 삼재검법 | 금 | 3번째 공격마다 3방향 검기 발사 |

## 적
### 쫄몹 유형
- **졸개**: 느리지만 다수, 근접
- **궁수**: 원거리 공격, 낮은 체력
- **무사**: 빠르고 강함, 소수
- **귀화(鬼火)**: 플레이어 추적하는 투사체형

### 보스 (스테이지별)
1. **구미호** - 분신 소환, 유혹 패턴
2. **장군귀** - 무거운 일격, 돌진
3. **독사대선** - 독안개, 소환
4. **업왕(業王)** - 최종 보스, 윤회의 근원

## 스테이지
- **1막: 폐촌** - 버려진 마을, 졸개/궁수 위주
- **2막: 대숲** - 빠른 적 위주, 시야 제한
- **3막: 폐사찰** - 귀화 다수, 함정
- **4막: 업왕의 사당** - 최종 보스

## 영구 성장 (메타 진행)
윤회(사망) 시 획득하는 "업(業)" 포인트로:
- 시작 체력 증가
- 새로운 무기 해금
- 새로운 증강 풀 해금
- 시작 증강 1개 선택 가능

## 기술 사양
- **해상도**: 1280x720 (16:9), 풀스크린 지원
- **도트 크기**: 기본 스프라이트 32x32 ~ 64x64
- **렌더링**: 2.5D (Y축 기반 깊이 정렬)
- **프레임**: 60fps 목표
- **타일 크기**: 16x16 기본 타일
